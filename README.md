# Epicyclic <img src="epilogodark.png" alt="Epicyclic logo" align="right" width="250" style="margin-left: 10px;"/>

Data from Planet Labs (3m/5m) is optimal for small-scale and epidemiological use case, however data bottlenecking creates barriers to easy analysis. Epicyclic, in tandem with *planknit*, seeks to create better PS/REScenes spatiotemporal capturing, delivery, and integrate cleanly into the Google Earth Engine and Cloud Storage ecosystems for rapid processing and analysis.
